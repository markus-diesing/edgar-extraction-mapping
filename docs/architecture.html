EDGAR Extraction & PRISM Mapping — System Architecture

LPA Internal POC · Architecture Diagram · March 2026

User Manual Tech Handbook

Overview

This tool is a local, standalone pipeline running entirely on a Mac. It retrieves SEC EDGAR 424B2 filings (structured products prospectuses), classifies them into PRISM payout types using a two-stage Claude AI pipeline, and extracts PRISM schema fields using a three-tier hybrid approach (registry metadata → HTML table parsing → LLM extraction). A browser-based UI enables human review, field correction, and export to JSON/CSV for downstream PRISM ingestion. All data is stored in a local SQLite database; no cloud infrastructure is required.
